# Speculative Commons

A single-page static website for **Speculative Commons** — a non-profit advocacy organisation that surfaces asymmetric philanthropic opportunities through rigorous analysis and independent thinking.

## Stack

- HTML, CSS, JavaScript (no frameworks)
- Google Fonts: EB Garamond + Inter
- Canvas-based particle background animation

## Local development

Open `index.html` in a browser — no build step required.

## Deployment

Deploy directly to **GitHub Pages** or **Vercel** by pointing at the root of this repository.
