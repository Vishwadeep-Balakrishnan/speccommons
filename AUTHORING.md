# Comprehensive Authoring Guide

## How to Add/Modify/Delete Essay Titles
1. **To Add**: Navigate to the `essays` directory and create a new file for the essay title. Make sure to name the file appropriately and update the index file to include this new title.
2. **To Modify**: Locate the file corresponding to the essay title you wish to change. Edit as required and save the changes.
3. **To Delete**: Remove the file associated with the essay title you want to delete and update the index file to reflect this change.

## How to Manage Tiles on the Homepage
- Edit the `homepage` configuration file to add or remove tiles. Each tile corresponds to a specific essay. Ensure the tiles reflect the current essays available.
- Update the layout files if necessary to accommodate new designs or formats for the tiles.

## How to Edit Essays
- Find the essay file in the `essays` directory. Open it in your text editor, make your modifications, and save.
- It's good practice to preview your changes before publishing them to ensure formatting is correct.

## How to Add Cover Images
1. **Upload Image**: Place the image file in the `/assets` directory.
2. **Link Image**: Edit the essay file to include the cover image by using the correct path to the image in the `assets` directory.

## How to Manage the /assets Directory
- Organize files by type (images, PDFs, etc.) and keep naming conventions consistent.
- Regularly audit the directory to eliminate unused files and maintain a clear structure to aid future management.
